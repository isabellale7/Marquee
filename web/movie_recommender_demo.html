How this works: the "Similar in style" tab vectorizes each film's genre tags with TF-IDF weighting and ranks the catalog by cosine similarity — computed live, right here in JavaScript. The "Audience also loved" tab is item-based collaborative filtering (counting what fans of this film also rated highly) — but the underlying ratings are from a simulated taste model, not real audience data, because the real MovieLens ratings file couldn't be reliably pulled through the tools available when this was built. See the project README for how to swap in the real dataset.
